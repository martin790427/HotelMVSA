# Hotel-MVSA-Quote-Tool 

This is a static front-end quote tool for Hotel MVSA — supports multi-room, 2 breakfast-price columns (no / with1&with2), and date-based rate categories.

## Deploy (GitHub Pages)
1. Create a new public repository on GitHub (e.g. `hotel-mvsa-quote-tool`).
2. Upload `index.html`, `style.css`, `script.js` to repository root.
3. Go to **Settings → Pages**, set **Deploy from a branch** → **main** → **/(root)**, Save.
4. After a minute your site will be available at `https://<your-username>.github.io/<repo-name>/`.

## Edit prices / rules
Open `script.js` and edit `PRICE_DATA` / `RULES` as needed.
